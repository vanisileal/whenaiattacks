**#When AI Attacks™ — AI Threat Assessment Workbooks**

Practitioner-grade threat assessments for the AI security landscape.
Each workbook takes a real, publicly-documented attack pattern and works it end to end: the operation, attacker and defender perspectives, a technical assessment mapped to OWASP, MITRE ATT&CK/ATLAS, and the Diamond Model, and a CISO debrief with concrete IR directives.

**#Who this is for**
CISOs, CSOs, security leaders, and the board — anyone accountable for navigating the AI threat landscape and translating it into decisions.

**#In this repository**
Non-Human Identities — machine credentials and AI agent identity governance (add each workbook as it's published)
Workbooks are published here as PDFs. The canonical articles live at https://whenaiattacks.ai/

These workbooks are free to download, use, and share. They're free on the website today, and they're free here.
